Navegação, listas e links

Navegação

Utiliza-se a tag "nav" para o bloco principal de links de navegação

Listas

Tag "ul" (agrupador de listas) e "li"(itens das listas) para listas não ordenadas

Tag "ol" (agrupador de listas) e "li" para listas ordenadas

Links

Tag "a href=" para conectar páginas no HTML, podendo ser internas páginas do nosso site) ou externas (páginas de outros sites)
